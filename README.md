# AIncient Labs Homebrew Tap

The official [Homebrew](https://brew.sh) tap for [AIncient Labs](https://github.com/aincient-labs).
Each formula in `Formula/` is installable as `aincient-labs/tap/<name>`:

```sh
brew install aincient-labs/tap/<formula>
```

## Formulae

| Formula | Description |
| ------- | ----------- |
| `atelier` | The `atelier` CLI — install, update, back up, and manage your [Atelier](https://github.com/aincient-labs/manager) CMS appliance. |

```sh
brew install aincient-labs/tap/atelier
```
